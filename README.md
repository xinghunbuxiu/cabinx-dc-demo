# cabinx-dc-demo
CabinX 动态组件使用示例

## 开始

### 前端项目

* 请将client内的前端项目复制到kayak-project文件夹下

#### 开始使用

1. 配置Host，前端设置的域名为 `rdcabinxdcdemo-partner.dmall.com`
```
127.0.0.1 rdcabinxdcdemo-partner.dmall.com
```

2. 启动
``` bash
cd path/your/kayak-project/cabinx_demo &&
yarn dev
```

### 后端项目

springboot的jar启动方式，导入IDE后直接启动
